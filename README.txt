
GameGenie Game Recommendation Program
--------------------------------------

Description:
This Python program offers personalized game recommendations based on user preferences. 
It's designed to help users discover new games within their preferred genres.

Features:
- Users can choose from a wide range of genres including action, crafting, racing, and more.
- The program randomly selects and recommends up to three games from the chosen genre.
- If the selected genre does not have any games, the user is informed accordingly.

How to Use:
1. Run the program.
2. When prompted, enter your preferred game genre.
3. The program will then display up to three game recommendations from that genre.

Note:
- Ensure Python is installed on your system to run this program.
- The game recommendations are based on a predefined list within the program.
